# Kasai — App gestione appartamenti

## Cos'è

App personale per gestire i propri appartamenti. Simile a Fixhos ma per uso personale di José.

## Stato

- Due versioni create: una con Claude Code, una con Codex
- Decisione: implementare le cose buone di Codex nella versione Claude Code

## Contesto
